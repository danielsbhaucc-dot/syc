# **App Name**: TroopSync

## Core Features:

- Template Management: Brigade-level users can create and manage templates defining required roles, authorizations, and equipment for different unit types using a drag-and-drop interface. This includes the ability to add, remove, and modify parameters within the templates.
- Automated Gap Analysis: The system automatically calculates and highlights discrepancies between the approved standard (defined by the brigade) and the actual status (reported by the battalion) across authorization, equipment, and personnel. It presents these gaps in a dashboard with visual alerts, such as color-coded warnings.
- Battalion Reporting: Battalion-level users fill out forms detailing their current status, including personnel, authorizations, and equipment. Data from the forms will populate fields within a dashboard. Includes options to input current status, like the number of soldiers with certain certifications, and available equipment like weapons.
- Role Assignment and Gap Identification: Brigade personnel assign soldiers to specific roles and equipment using a form or dashboard. The system then automatically checks for discrepancies between assigned roles, authorizations, and equipment, highlighting any gaps based on predefined rules and data validation.
- User Authentication and Authorization: Secure login for brigade and battalion users, as well as a one-time access code for view-only access. User roles and permissions are strictly enforced, controlling access to features and data based on the user's role.
- Dashboard Views: Brigade-level users gain access to a real-time dashboard showing a unit readiness status. Battalion users view only their respective information.
- Data Export: Users can export formatted reports in PDF and Excel formats. Exported PDF is designed for sharing. The exported excel sheet supports deeper dives into collected data. Reports also provide historical data to compare between time periods.

## Style Guidelines:

- Primary color: Deep blue (#3B82F6) to convey trust and stability, referencing the tactical nature of the app.
- Background color: Dark slate (#1E293B), providing a modern look with sufficient contrast to reduce eye strain in dim environments.
- Accent color: Emerald green (#34D399) to highlight key metrics, indicate positive status, and suggest the real-world association with field operations and technology.
- Headline font: 'Space Grotesk' sans-serif for headlines and titles, for a tech-oriented modern feel.
- Body font: 'Inter' sans-serif for body text, known for its clear and readable design.
- Font Awesome Free 6.4 for tactical symbols.
- Card-based design for presenting units and personnel with at-a-glance key stats, coupled with color-coded warnings and alerts to focus user attention.
