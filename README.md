# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Deployment to a Non-Google Cloud Server

To deploy this Next.js project to a non-Google Cloud server, you will first need to build the project. This can typically be done by running `npm run build` or `yarn build` in your project's root directory. After the build process completes, you can then either serve the static output (if your application is configured for static export) or run the Next.js server to host your server-side rendered application.
